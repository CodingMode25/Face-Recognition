# Simple OpenCV Face Recognition in Python


### Prerequisites
* Python 3.6.4
* OpenCV 3.4.1 or opencv-contrib-python 4.0
* Numpy
* Pillow

### Installing

* Download Above Repository 

## Running the tests

* run the dataSetGenerator.py and enter a unique id to create face samples with your face
* run trainer.py
* run detector.py
